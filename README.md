# QingYu's Blog

一个基于 Hexo 的个人博客项目，当前风格为文档化阅读体验：

- 首页有头图与标签筛选
- 文章页目录可跟随阅读位置
- 主题已做轻量化定制

## 本地开发

```bash
npm install
npm run server
```

默认访问地址：

http://localhost:4000/

## 常用命令

```bash
npm run clean
npm run build
npm run server
```

## 目录结构

- `source/_posts/`：文章内容
- `source/css/custom.css`：自定义样式
- `source/js/docs-interactive.js`：交互逻辑
- `themes/landscape/`：主题模板与配置
